# My C++ Learning Portfolio

Структурированное портфолио учебных проектов на C++.

## Структура
- **basics/** — учебные программы
- **projects/** — pet-проекты (в процессе).

## Сборка
`g++ -std=c++11 путь/к/файлу.cpp -o имя_программы`

### О проектах
1. Консольная программа для учёта автомобилей (список машин в Forza Horizon 5).  
- Данные хранятся в `std::vector` и сохраняются в текстовый файл `garage.txt`.  
- При запуске файл загружается, при выходе — сохраняется.  
- Реализованы функции добавления, просмотра и защита от некорректного ввода.
2. Программа решает квадратное уравнение вида `ax^2+bx+c=0`.  
- Парсит строку, введенную пользователем (удаляет пробелы, ищет `=`, `x^2`, `x`).  
- Вычисляет дискриминант и выводит корни (вещественные или сообщение об отсутствии).  
- Защита от некорректного ввода и проверка формата.
